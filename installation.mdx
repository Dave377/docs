Coreframe is distributed through the Figma Community plugin marketplace. Installation takes less than a minute and requires no additional software — just a Figma account and a file to work in. You can install it directly from the Resources panel inside Figma or from the Community website.

## Install from Figma Community

<Steps>
  <Step title="Open Figma or FigJam">
    Launch the Figma desktop app or open [figma.com](https://www.figma.com) in your browser. You can install the plugin from any file, including a new blank file.
  </Step>
  <Step title="Open the Resources panel">
    Press **Shift + I** to open the Resources panel, then click the **Plugins** tab. This shows all available and previously installed plugins.
  </Step>
  <Step title="Search for Coreframe">
    In the search field, type **Coreframe**. The plugin will appear in the results list.
  </Step>
  <Step title="Install the plugin">
    Click the **Install** button next to Coreframe. The button changes to a checkmark once installation is complete. The plugin is now available across all your Figma files.
  </Step>
</Steps>

## Open the plugin

Once installed, you can launch Coreframe from anywhere in Figma:

<Steps>
  <Step title="From the main menu">
    Go to your canvas → click the **Figma logo** in the top-left corner → select **Plugins** → **Coreframe**.
  </Step>
  <Step title="From the canvas right-click menu">
    Right-click anywhere on the canvas → select **Plugins** → **Coreframe**. This is the fastest way to re-open the plugin while you're working.
  </Step>
</Steps>

## System requirements

- Figma Desktop (Mac or Windows) or Figma in any modern browser
- Any operating system — Windows, macOS, or Linux
- No additional software, runtimes, or local dependencies required

<Note>
  **Plugin ID:** `1646208658384992348`

  If you need to add Coreframe by ID — for example, to install it in an organization's approved plugin list or via Figma's plugin management settings — use the identifier above.
</Note>

Once Coreframe is installed, follow the [Quickstart](./quickstart) guide to generate your first design system tokens and components.
