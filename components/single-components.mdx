The Single Component Generator lets you drop any Coreframe component directly onto your active Figma canvas without touching the rest of your library. Instead of regenerating the full component set, you pick exactly the component — and the specific sub-variant or state — you need right now, and Coreframe places it where you're working. It's the fastest way to grab a single button, a pre-built Hero Section, or any other piece of the system mid-design.

## Using the Single Component Generator

<Steps>
  <Step title="Open the Single tab">
    In the Coreframe plugin panel, navigate to the **Single** tab (also accessible from the Components tab by switching to Single mode). You'll see the same searchable grid used in the full Components view.
  </Step>
  <Step title="Find the component you need">
    Type in the search bar or use the category filter to narrow the list. Categories include Forms, Navigation, Feedback, Data Display, Overlays, Layout, Marketing, and Ecommerce. Each card shows a thumbnail preview of that specific sub-variant.
  </Step>
  <Step title="Click the component to select it">
    Click a component card to highlight it. The selected state is indicated by a violet border and a filled dot in the card footer.
  </Step>
  <Step title="Click Insert">
    Click the **Insert** button (or click directly on the hovered card's insert overlay). Coreframe places the component as a frame on your current Figma canvas, at the center of your viewport.
  </Step>
</Steps>

<Note>
  Single components are placed as instances of the Coreframe component library. If the base component set doesn't exist in your file yet, Coreframe will generate it automatically before placing the instance.
</Note>

## Available single components

The Single Component Generator exposes a wider set of granular sub-variants that aren't surfaced individually in the full library grid. In addition to the standard components, you can insert:

- **Button variants** — Button Primary, Button Outline, Button With Icon, Button Icon, Button Group
- **Avatar variants** — Avatar Initials, Avatar Image, Avatar Icon, Avatar Status Badge, Avatar Group
- **Chip variants** — Chip, Chip With Dot, Chip With Icon
- **Tooltip directions** — Tooltip Top, Tooltip Right, Tooltip Bottom, Tooltip Left
- **Toggle and checkbox states** — Toggle On, Toggle Off, Checkbox Checked, Checkbox Unchecked, Radio Checked, Radio Unchecked
- **Social auth buttons** — Google Button, Apple Button, Facebook Button, GitHub Button, LinkedIn Button
- **Marketing & Ecommerce sections** — Hero Section, Blog Card, Testimonial Card, Product Card
- **Form sections** — Sign Up
- **Navigation utilities** — Navbar, Mega Menu, Nav Pills, Dropdown Menu, Breadcrumb, Pagination, Pagination Item, Tabs
- **Feedback utilities** — Progress, Progress Stepper, Step Indicator, Spinner, Rating
- **Data display utilities** — Table, Stat, Section Title, Pill, Accordion, Card, Avatar Group
- **Layout utilities** — Content Divider, Input Range, Dropdown Open

These single-insert variants give you precise control when you need a specific state or composition without pulling in the full variant matrix.
